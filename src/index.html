Restaurant Web Design Concept

This is one of my favorite projects. The restaurant is one of my favorites in the area and I decided it would be a cool idea to recreate the home page. The design was built using Bootstrap to ensure responsiveness across all platforms. One of the key features of this project is the image gallery.
